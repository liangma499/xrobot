消息参数：

<table>
    <tr>
        <th>字段</th>

    </tr>

        <td>object</td>
        <td>-</td>

    </tr>

        <td>string</td>

    </tr>
    <tr>
        <td>currRound.countdown</td>
        <td>float</td>
        <td>-</td>
        <td>进入下一个状态的倒计时（秒），飞行状态下无此参数</td>
    </tr>
    <tr>
        <td>currRound.flightDuration</td>
        <td>float</td>
        <td>-</td>
        <td>飞行时长（秒），仅在飞行状态下有此参数</td>
    </tr>
    <tr>
        <td>currRound.crashMultiple</td>
        <td>float</td>
        <td>-</td>
        <td>坠毁倍率</td>
    </tr>
    <tr>
        <td>nextRound</td>
        <td>object</td>
        <td>-</td>
        <td>当前局信息</td>
    </tr>
    <tr>
        <td>nextRound.no</td>
        <td>string</td>
        <td>-</td>
        <td>局编号</td>
    </tr>
    <tr>
        <td>nextRound.state</td>
        <td>int</td>
        <td>-</td>
        <td>局状态（0：未开始 1：押注中 2：飞行中 3：坠毁）</td>
    </tr>
    <tr>
        <td>nextRound.countdown</td>
        <td>float</td>
        <td>-</td>
        <td>进入下一个状态的倒计时（秒），飞行状态下无此参数</td>
    </tr>
</table>
